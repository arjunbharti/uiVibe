Avatar

The Avatar component can be used to represent a user's display picture, initials, etc

Image Avatar

Image avatar can be used with three different sizes as per your use-case. The three different sizes are normal "avatar", smaller "avatar-sm" and larger "avatar-lg"

Use class name as "avatar" in your "img" tag as mentioned below: 
                    <img class="avatar" src="img-url-xyz" alt="avatar" />
                        
<img class="avatar avatar-sm" src="img-url-xyz" alt="avatar" />
                        
<img class="avatar avatar-lg" src="img-url-xyz" alt="avatar" />

Letter Avatar

Letter avatar can be created using avatar avatar-text with three different sizes in the div tag.

AB
AB
AB
Use class name as "avatar-text" in your "div" tag as mentioned below: 
                    <div class="avatar avatar-text avatar-sm">AB</div>

<div class="avatar avatar-text ">AB</div>
                        
<div class="avatar avatar-text avatar-lg">AB</div>

Button

The Button component is used to trigger different actions or events, such as submitting a form, performing delete operation, cancelling an action, etc.

Standard Buttons

To use primary and secondary buttons, write "btn-primary" and "btn-secondary" respectively in the class name. For curved shape button, use "btn-curved" in the class name.

Use the code snippets mentioned below: 
                        <button class="btn btn-primary">Primary</button>
                            
<button class="btn btn-primary btn-curved">Primary</button>
                            
<button class="btn btn-primary" disabled>Disabled</button>
                            
<button class="btn btn-secondary">Secondary</button>
                            
<button class="btn btn-secondary btn-curved">Secondary</button>
                            
<button class="btn btn-secondary" disabled>Disabled</button>

Link Buttons

To use link buttons, write "btn-link" in the class name.

Link Button Link Button
Use the code snippets mentioned below: 
                        <a href="#" class="btn btn-link">Link Button</a>
                            
<a href="#" class="btn btn-curved btn-link">Link Button</a>

Icon Buttons

To use icon buttons, use "btn-icon" class name in the button tag

Use the code snippets mentioned below: 
                        <button class="btn btn-icon">

Floating Action Buttons
